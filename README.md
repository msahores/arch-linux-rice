**Graphical Environment:**
X.org(X)
*To install it:*
```
pacman -S xorg-server xorg-xinit
```

*To run it:*
```
echo "startx" >> .xinitrc
```

I have also a spanish keyboard layout for xorg setted in .xinitrc
Change it there for the required one.
